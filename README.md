<br>
An instructor-led introductory notebook (_Jupyter_ notebook), NRSC7610_intro.ipynb, is the place to start. The class will all begin by running this notebook together, including the instructor, to get a feel for the basic syntax of Python and the flow of using a Jupyter notebook for data exploration and analysis. We won't use any LLM tools for this section - not because these tools are cheating or because they don't work. In general, such tools ([Codex](https://openai.com/codex/), [Claude](https://code.claude.com/docs/en/overview), [Copilot](https://github.com/features/copilot)) are very useful, but effective use benefits from understanding of basic syntax. Students will complete the remainder of the Intro notebook in small groups or on their own, to develop confidence in independent coding and syntax understanding. Please refrain from code-assist tools at this stage - but don't worry, shackles off later.

### optional extension and practice
After we go through the intro notebook together in class, you *can* choose one (or more!) of the exercise notebooks to work through. These notebooks will have a mixed of pre-written code and exercises, always towards an analysis goal typical for the modality. The four exercise notebook options are:
1. Direction Tuning Analysis
2. Time Series (Patch Clamp Vm) Analysis
3. Fluorenscene Image Analysis
